== Description ==

WP-Mon is a really simple WordPress-Monitor.
You can see you RAM- & Disk-Status, any PHP-Info, important infos at one page and more.

== Installation ==

1.) Install it from wordpress.org or upload it
2.) Activate it
3.) Read the changelog and setup the plugin
4.) Monitoring you WP-Installation
